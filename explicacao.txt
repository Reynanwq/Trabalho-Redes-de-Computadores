FUNÇÕES:

- Listar arquivos disponíveis para um cliente.
- Permitir que um cliente deposite um arquivo no servirdor.
- Permitir que o cliente recupere um arquivo do servidor.
- Permite que o cliente exclua um arquivo do servidor.

-------------------------- DESCRIÇÃO DE CADA OPERAÇÃO --------------------------

DEPÓSITO:
O cliente informa ao servidor o seu nome, a quantidade de cópias que deseja armazenar, o nome do arquivo (incluindo o tipo: .txt; .js...) e o conteúdo do arquivo.

RECUPERAÇÃO:
O cliente envia uma solicitação ao servidor especificando o nome do arquivo que deseja recuperar. O servidor procura uma das réplicas do arquivo nas pastas onde foi depositado; se encontrar, o servidor responde ao cliente com o conteúdo do arquivo recuperado.

REMOÇÃO:
O cliente indica o seu nome e o nome do arquivo que deseja remover. O servidor mapeia e realiza uma busca no diretório principal. Caso encontre a pasta com o nome do cliente, ele realiza outra busca dentro da pasta do cliente, procurando o arquivo que deseja excluir.

LISTAR:
O cliente informa o seu nome e o servidor mapeia e realiza uma busca no diretório principal. Se encontrar a pasta com o nome do cliente, ele realiza outra busca dentro da pasta do cliente, guardando os nomes dos arquivos contidos nela e enviando esses nomes para o cliente.
