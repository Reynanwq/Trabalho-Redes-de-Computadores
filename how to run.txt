-------------- DEPÊNDENCIAS --------------

Para instalar as dependências, abra o terminal e digite:

npm install 
npm install socket.io-client
npm install dotenv

Provavelmente, é para está exectudando normalmente. Se não executar, entre em contato:

1° Email: reynanwq@gmail.com
2° Email: reynanpaiva@ufba.br

-------------- EXECUTANDO --------------

Abra 2 terminais:
No primeiro terminal digite "node server.js"
No segundo terminal digite "node cliente.js"

Para testar com os espelhos, abra 3 ou mais terminal e digite o "node server.js"

-------------- EXECUTANDO DEPOSIT --------------

no mesmo terminal em que digitou "node cliente.js"
insira o comando:

deposit NomeDoCliente NomeDoArquivo

exemplo: deposit Reynan arquivoTeste1.txt

-------------- EXECUTANDO RECOVER --------------

no mesmo terminal em que digitou "node cliente.js"
insira o comando:

recover NomeDoCliente NomeDoArquivo NomeDoArquivoQueDesejaVerOconteudo

exemplo: recover Reynan arquivoTeste1.txt recuperado.txt


-------------- EXECUTANDO DELETE --------------

no mesmo terminal em que digitou "node cliente.js"
insira o comando:

delete NomeDoCliente NomeDoArquivo

exemplo: delete Reynan arquivoTeste1.txt

-------------- EXECUTANDO LIST --------------

no mesmo terminal em que digitou "node cliente.js"
insira o comando:

list NomeDoCliente 

exemplo: list Reynan
